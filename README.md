# Climate-Change-and-Global-Food-Security
Climate change has become one of the most significant global challenges, influencing environmental, economic and social systems.

This project explores how climate indicators, natural disasters, agricultural production, international trade and food prices evolved between 2000 and 2024 using publicly available datasets and an interactive Power BI dashboard.

The objective is not to establish causal relationships, but to identify trends and possible connections across multiple domains.
